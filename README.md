## 🛠 Технологии и инструменты

Проект построен с использованием следующих инструментов:

- **Сборщик:** [Gulp](https://gulpjs.com)
- **Препроцессор:** [SASS/SCSS](https://sass-lang.com)
- **Развертывание:** Автоматический деплой на GitHub Pages через GitHub Actions при пуше в ветку `main`.

## 🚀 Быстрый старт

Версия Node для сборки проекта [24.14.1](https://nodejs.org/en/blog/release/v24.14.1).

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/NVinox/sveak-frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустить команду:
   ```bash
   npm run dev
   ```

## Доступные команды

- `build` - Сборка проекта. Соберет картинки, html, scss, js файлы в `build`;
- `dev` - Запуск в режиме разработки. Воспроизводит команду `build` + старт проекта на локальном сервере.
